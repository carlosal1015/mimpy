Class for constructing structured hexahedral meshes.

build_mesh(ni, nj, nk, dim_x, dim_y, dim_z, K, modification_function=None)[source]¶

Constructs a structured hexahedral mesh.

Parameters:
  • ni (int) – Number of cells in the x-direction.
  • nj (int) – Number of cells in the y-direction.
  • nk (int) – Number of cells in the z-direction.
  • K (function) – Permeability map function. K(point, i, j, k ) -> 3x3 Matrix.
  • dim_x (float) – Size of domain in the x-direction.
  • dim_y (float) – Size of domain in the y-direction.
  • dim_z (float) – Size of domain in the z-direction.
  • modification_function (function) – Alteration function for shifting points of the cells. modification_function(p) -> 3 array.
Returns:

None
